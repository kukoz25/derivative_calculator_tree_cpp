◆ upraszczanie()

string upraszczanie ( drzewo *  w)

Funkcja upraszcza drzewo gałęzie drzewa

Parametry
drzewopotrzebne jest w ogóle drzewo by je można było skrócić
Zwraca
zwaraca funckje maksymalnie skróconą

jeśli część z lewej strony jest zerem to zapisuje tylko prawą część

jeśli część z prawej strony jest zerem to zapisuje tylko lewą część

jeśli część z lewej strony jest zerem to zapisuje tylko prawą część z minusem

jeśli część z prawej strony jest zerem to zapisuje tylko lewą część

jeśli którekolwiek jest 0 to wynik jest 0

jesli ktoraś galaz jest jedynka to zapisujemy tylko lewą/prawą część

jeśli licznik jest 0 to calość jest 0

jesli mianownik jest 1 to wypisz tylko licznik
